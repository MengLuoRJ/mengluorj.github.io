        # 地图导航

        提示

        可以通过点击每张地图所附的 battlenet::// 链接来在游戏中直接建立房间。

        如果尚无 battlenet 链接支持，可以将所附链接复制至游戏聊天区中，发送后点击来建立房间。

        # 1 - 自由曰

        雷诺的游骑兵正在袭击一个玛·萨拉上无辜的帝国前哨站，确保他不会破坏帝国在这个区域的权威。

        • 链接：battlenet:://starcraft/map/5/110115

        # 2 - 不法之徒

        雷诺正在玛·萨拉上尝试偷取蒙斯克辛苦得来的神器。确保他和他的叛军盟友无法得逞！

        • 链接：battlenet:://starcraft/map/5/110236

        # 3 - 零点行动

        凯瑞甘发现了在玛·萨拉的雷诺，并想给他一个拥抱。但出于一些“深不可测”的原因，雷诺拒绝了。是时候去给他上一课了！

        • 链接：battlenet:://starcraft/map/5/110243

        # 4 - 来之不易

        塔达林执政官奈姆，十分喜爱他那闪亮的神器。但雷诺和凯瑞甘想得到它。阻止他们偷取神器！

        • 链接：battlenet:://starcraft/map/5/110265

        # 5 - 大撤离

        汉森喜爱她的殖民队，但凯瑞甘并不这么想。雷诺对汉森有所好感，并试图救出她，但凯瑞甘并不没有。

        • 链接：battlenet:://starcraft/map/5/110290

        # 6 - 大爆发

        你是个感染者，而这里正好有个殖民地。快开始你的工作！

        • 链接：battlenet:://starcraft/map/5/110324

        # 7 - 恶魔游乐场

        雷诺受托什邀约去红石星为其采集丰富的晶体矿，而虫群在此不断地惹恼他。

        • 链接：battlenet:://starcraft/map/5/110393

        # 8 - 火车大劫案

        霍纳发现了一些正在执行回收行动的火车。阻止他们破化你的轨道，帝国不希望自己的回收物资被抢走。

        • 链接：battlenet:://starcraft/map/5/110391

        # 9 - 挖宝行动

        塔达林有一个超酷的神器，而雷诺的游骑兵“得到了”一台超酷的激光钻机。让我们看看哪个更酷！

        • 链接：battlenet:://starcraft/map/5/110392

        # 10 - 末日密语

        泽拉图在乌兰发现了一个上古预言。凯瑞甘并不希望他将此散播开来。让他闭嘴，好吗？

        • 链接：battlenet:://starcraft/map/5/110458

        # 11A - 拯救海文

        异虫病毒的感染已经侵袭海文，而雷诺的游骑兵却盲目的为汉森而战。做些什么，阻止异虫的威胁。

        • 链接：battlenet:://starcraft/map/5/110459

        # 11B - 海文的陷落

        雷诺睁大双眼，意识到绝大多数的移民都已经被感染！但他为什么不让星灵帮忙……

        • 链接：battlenet:://starcraft/map/5/110460

        # 12 - 恶兆

        泽拉图有一个预言需要解读，而那些能够揭示这个预言的人都被囚禁在这里。玛尔将确保他们被永远深埋于此！

        • 链接：battlenet:://starcraft/map/5/110477

        # 13 - 博弈

        奥尔兰与帝国达成了一项不错的交易。雷诺的游骑兵们对此并不太满意……

        • 链接：battlenet:://starcraft/map/5/110478

        # 14 - 未来回响

        泽拉图想要亵渎异虫最为神圣的……东西，主宰（的遗骸）。你这一生只有一个目标，捍卫你那（死去的）主人。

        • 链接：battlenet:://starcraft/map/5/111873
